# Risk Register

| Risk Category | Description | Impact | Mitigation |
|--------------|------------|--------|------------|
| Bias | Model may reflect biased outputs if trained on real data | Medium | Use diverse datasets and fairness checks |
| Robustness | Model may fail on unexpected inputs | Medium | Add input validation and anomaly detection |
| Privacy | Potential exposure of sensitive data | Low | Avoid using real user data |
| Compliance | Model may not meet regulatory standards | Medium | Follow governance frameworks and documentation |
